# Software Package Database

This project implements a software package database management system.

## Build Instructions

The project has been successfully built. To build again, navigate to the database_project directory and run:

```bash
cd /workspace/database_project
make clean
make
```

## Running the Application

To run the application:

```bash
cd /workspace/database_project
./SoftwarePackageDatabase
```

Or use the helper script:

```bash
/workspace/run_app.sh
```

Note: The application requires a GUI environment. If running in a headless environment, you'll need to set up X11 forwarding or run in a GUI-enabled environment.
